> - N을 입력받은 뒤 N의 개수만큼 오름차순으로 정렬하기 위하여 퀵 정렬**[O(NlogN)]**을 사용하였다.

>   *이 문제에서는 키 값이 클 수도 있기 때문에 시간 복잡도가 O(NlogN)이 아니게 되지만 그냥 퀵 정렬을 사용하기로 하였다.*
